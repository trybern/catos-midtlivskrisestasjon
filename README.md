# Catos-midtlivskrisestasjon 🌴🍹

En humoristisk og stilfull "mobile-first" nettside der man kan bestille sin fremtidige unge kjæreste. Designet er inspirert av Mid-Century Modern Tiki-lounger fra 50-tallet.

## Innhold
Prosjektet er strippet ned til det essensielle og krever ingen bygge-steg (build steps):
- `index.html` - Nettsidens struktur og innhold.
- `style.css` - Alt av design, variabler og grid-system.
- `script.js` - Funksjonalitet for å hente utjemaskjema-data og klargjøre bestillings-e-posten.

## Hvordan bruke det
Siden dette er en statisk nettside, trenger du ingen server eller database. Du kan bare åpne `index.html` direkte i hvilken som helst nettleser!

Når man trykker på "Klar for krisen!", trigges klientens standard e-postprogram via en `mailto:`-lenke.

## Teknisk
- Bygget med ren HTML, verken React eller backend-systemer.
- Vanilla CSS, bruker moderne CSS Grid.
- Vanilla JavaScript (ES6+).
- Fonter hostes fra Google Fonts (Abril Fatface og Outfit).
